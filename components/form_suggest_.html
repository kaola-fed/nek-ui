Suggest

Kind: global class
Extend: Dropdown

new Suggest()

Param Type Default Description
[options.data] object = 绑定属性
[options.data.source] Array.<object> [] <=> 数据源
[options.data.source[].name] string => 每项的内容
[options.data.key] string "id" => 数据项的键
[options.data.nameKey] string "name" => 数据项的name键
[options.data.source[].disabled] boolean false => 禁用此项
[options.data.selected] object <=> 当前选择项
[options.data.value] string <=> 文本框中的值
[options.data.id] string <=> 选项的id值;
[options.data.placeholder] string "请输入" => 文本框的占位文字
[options.data.maxlength] number => 文本框的最大长度
[options.data.startLength] number 0 => 开始提示长度。当输入长度>=该值后开始提示
[options.data.matchType] string "all" => 匹配方式，all表示匹配全局，start表示只匹配开头，end表示只匹配结尾
[options.data.strict] boolean => 是否为严格模式。当为严格模式时，value属性必须在source中选择，否则为空。
[options.data.autofocus] boolean => 是否自动获得焦点
[options.data.itemTemplate] string null @=> 单项模板
[options.data.hideTip] boolean false => 是否显示校验错误信息
[options.data.open] boolean <=> 当前为展开/收起状态
[options.data.readonly] boolean => 是否只读
[options.data.disabled] boolean => 是否禁用
[options.data.visible] boolean true => 是否显示
[options.data.class] string => 补充class
[options.service] object @=> 数据服务
